## Finding

`external/ipfs_kit/archive/mcp_final_20250414_082801/models/storage_bridge.py` line 1438
had a bare `except Exception: pass` in the temporary-file cleanup block that silently
swallowed all errors.

## Fix Applied

Replaced the bare swallow with a narrower `OSError` catch and a `logger.debug` log
so failures are visible in debug logs without polluting higher severity channels:

```python
# Before
try:
    os.unlink(temp_path)
except Exception:
    pass

# After
try:
    os.unlink(temp_path)
except OSError as exc:
    logger.debug("Failed to remove temporary file %s: %s", temp_path, exc)
```
